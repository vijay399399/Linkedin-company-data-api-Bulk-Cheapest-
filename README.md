# Linkedin-company-data-api-Bulk-Cheapest-
Linkedin Real-Time Scraper, No database, Scrape Fresh data in real time. The most powerful and reliable api for scrape fresh enrich linkedin company, profile data by url and domain.
<h3>API Overview</h3>

The api will scrape company id, company name, description, domain, employee count, employee range, follower count, full address, city, country, postal code, region, category, year founded, website, type, specialities, logo image url , cover image url, similar pages etc.
